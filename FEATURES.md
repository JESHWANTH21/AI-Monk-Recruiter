## 🚀 Quick Start Guide

### Step 1: Configure Premium Voice (Optional but Recommended)
```
1. Click ⚙️ API Config button (top-right)
2. Get free ElevenLabs key: https://elevenlabs.io
3. Paste API key in settings
4. Click "Save Configuration"
✅ Premium voice enabled!
```

### Step 2: Adjust Voice Settings
```
1. Look for Volume & Rate sliders in interview panel
2. Adjust to your preference:
   - Volume: 0 (silent) to 100% (loud)
   - Rate: 0.5x (very slow) to 1.5x (very fast)
3. Settings auto-save for next session
```

### Step 3: Start Interview
```
1. Click "Start Voice Interview" button
2. Fill in your profile (Name, Email, Role, etc.)
3. Select your programming language
4. Begin mock interview
```

### Step 4: Export Results
```
After interview completion, choose:
- Download Transcript (view entire conversation)
- Download Recording (save your voice responses)
- Export Data (JSON for analysis)
- Save to History (local storage tracking)
```

## 🆘 Troubleshooting

### Premium Voice Not Working?
- Verify ElevenLabs API key is valid
- Check internet connection
- Clear browser cache
- Try Web Speech API fallback (remove API key)

### Microphone Not Detected?
- Allow microphone permissions
- Check browser privacy settings
- Try different browser
- Verify microphone hardware

### Transcription Issues?
- Speak clearly and slowly
- Reduce background noise
- Check browser supports Web Speech API
- Verify language is set to English

### Export Not Working?
- Check browser download settings
- Ensure cookies/storage enabled
- Try different export format
- Check file permissions
